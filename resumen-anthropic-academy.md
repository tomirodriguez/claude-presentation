# Resumen: Anthropic Academy - Claude Code Course

**Fuente**: [Anthropic Academy](https://academy.anthropic.com/)

---

## Descripción General

Anthropic Academy ofrece un curso estructurado para aprender Claude Code desde lo básico hasta técnicas avanzadas. Es el recurso oficial de aprendizaje de Anthropic.

---

## Estructura del Curso

### 8 Áreas de Aprendizaje

1. **Introduction to Claude Code**
   - Qué es y por qué usarlo
   - Instalación y setup
   - Primera sesión

2. **Basic Operations**
   - Comandos esenciales
   - Lectura y escritura de archivos
   - Ejecución de comandos

3. **Context Management**
   - Ventana de contexto
   - `/clear` y `/compact`
   - Estrategias de sesión

4. **CLAUDE.md Configuration**
   - Estructura del archivo
   - Jerarquía de configuración
   - Best practices

5. **Custom Commands**
   - Slash commands personalizados
   - Frontmatter y opciones
   - Organización por namespaces

6. **Advanced Features**
   - Hooks
   - Skills
   - Subagentes

7. **MCP Integration**
   - Model Context Protocol
   - Servidores MCP
   - Extensibilidad

8. **Best Practices & Workflows**
   - Patrones de trabajo efectivos
   - Anti-patterns a evitar
   - Casos de uso avanzados

## Temas Destacados

### Context Management (Lección Clave)

```
- Por qué el contexto importa
- Límites de la ventana de contexto
- Cuándo y cómo limpiar
- Técnicas de compactación efectiva
```

### CLAUDE.md (Lección Fundamental)

```
- Anatomía del archivo
- Secciones recomendadas
- Ejemplos por tipo de proyecto
- Integración con el equipo
```

### Workflows Agentic (Lección Avanzada)

```
- Explore → Plan → Code → Commit
- Uso de Plan Mode
- Subagentes para trabajo paralelo
- Automatización con hooks
```
